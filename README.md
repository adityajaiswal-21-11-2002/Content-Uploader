# Content Upload Tracker

A Next.js application for tracking content uploads and employee compliance.

## Features

- Daily upload tracking
- Weekly progress monitoring
- Monthly analytics and leaderboards
- Employee activity tracking
- Compliance monitoring
- Topic suggestions and generation
- Email alerts and notifications

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Set up environment variables:
Create a `.env` file with your configuration (see `.env.example` if available)

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
